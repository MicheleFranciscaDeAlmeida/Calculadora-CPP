# Calculadora em C++

## 📝 Descrição
Projeto de uma calculadora básica em C++, utilizando **Programação Orientada a Objetos (OOP)**. 
Permite realizar operações matemáticas fundamentais com classes e métodos, demonstrando conceitos de **classes, objetos, encapsulamento e métodos**.

## ⚙️ Funcionalidades
- Soma, Subtração, Multiplicação e Divisão
- Estrutura orientada a objetos
- Tratamento de entradas inválidas

## 💻 Tecnologias
- Linguagem: C++
- Compilador: G++ / GCC
- Conceitos aplicados: OOP, classes, métodos, condicionais

## 🚀 Como executar
1. Instale o G++:  
   - Windows: [MinGW](https://www.mingw-w64.org/)  
   - Linux: `sudo apt install g++`  
   - MacOS: `brew install gcc`  

2. Clone o repositório:  
```bash
git clone https://github.com/SEU-USUARIO/calculadora-cpp.git
cd calculadora-cpp/src

3. Compile e Execute:
g++ main.cpp Calculadora.cpp -o calculadora
./calculadora
